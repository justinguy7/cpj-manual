# Software Overview

### Swift GCS

![Swift GCS Logo](img/swiftgcs-logo.png)

Swift Ground Control Station (GCS) is the UAS user interface. The standard Sapphire package includes one Swift GCS license which is good for two activations.  

### ArduPilot

![Ardupilot logo](img/ardupilot-logo.png)

ArduPilot is an open-source autopilot firmware for flying UAS. ArduPilot is distributed under a GPLv3 license, and the source code is publicly available.

### Skylink

![Trillium Logo](img/trillium.png)

[Skylink](https://www.trilliumeng.com/technology/skylink-software) is the Trillium gimbal user interface.
