## Building locally

To work locally with this project, you'll have to follow the steps below:

1. Fork, clone or download this project
1. Install honkit `npm install honkit`
1. Install plugins `npm install gitbook-plugin-hints`
1. Preview your project: `npx honkit serve`
1. Add content
1. Generate the website: `honkit build` (optional)
1. Push your changes to the master branch: `git push`

Read more at HonKit's [documentation][https://honkit.netlify.app/].
